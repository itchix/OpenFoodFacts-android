Para alimentos de humanos, você pode instalar Open Food Facts, para cosméticos, Open Beauty Facts e, para todos os outros produtos, Open Products Facts.
But what about other products ? O Open Products Facts é um banco de dados aberto de todos os produtos do mundo.

***Contribua para Transparência de Alimentos para Animais***

- Open Pet Food Facts é um banco de dados com produtos alimentícios para animais, feito por todo mundo, para todo mundo.
- Você pode utilizá-lo para fazer melhores escolhas de alimentos para animais e, por ser dados abertos, qualquer um pode reutilizar para qualquer proposta.
- Open Pet Food Facts é um projeto sem fins lucrativos desenvolvido por milhares de voluntários em todo o mundo. Temos muitos projetos interessantes para os quais você pode contribuir de diversas maneiras. 
- Você pode começar a contribuir hoje, adicionando informações de um produto do seu pet que esteja no armário.

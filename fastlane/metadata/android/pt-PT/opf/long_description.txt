Para alimentação humana, pode instalar o Open Food Facts, para cosméticos, o Open Beauty Facts e para todos os outros produtos, o Open Products Facts.
Mas e os outros produtos? O Open Food Facts é uma base de dados de produtos alimentares com ingredientes, alergénios, tabela nutricional e todos os pedaços de informação que se encontram nos rótulos dos produtos.

***Contribuir para a transparência dos alimentos para animais ***

- O Open Pet Food Facts é uma base de dados de produtos alimentares para animais feito por todos, para todos.
- Pode usá-lo para fazer melhores escolhas alimentares para animais e como são dados abertos, qualquer um pode reutilizá-lo para qualquer finalidade.
- O Open Pet Food Facts é um projeto sem fins lucrativos desenvolvido por milhares de voluntários em todo o mundo. Temos muitos projetos interessantes para os quais pode contribuir de diversas maneiras. 
- Pode começar a contribuir hoje, adicionando um produto do seu armário de alimentos para animais.
